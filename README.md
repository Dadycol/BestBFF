<h1>Para BFF</h1>

<p>Este projeto contém um simples pedido de brincadeira feito em HTML e JS, com um toque divertido. Literalmente irresistível, quando o mouse se aproxima do botão "não", ele "foge" aparecendo aleatoriamente na tela, impossibilitando clicar no "não". Se o usuário clicar no botão "Sim", ele será surpreendido com uma mensagem de congratulações</p>
